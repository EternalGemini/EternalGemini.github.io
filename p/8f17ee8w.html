CS上线

2022-02-14
#仅供学习参考！！！

Cobalt Strike 一款以由美国redteam团队，研发出来的一款以Metasploit为基础的GUI框架式渗透测试工具。

一、搭建服务器

1.在kali中解压压缩包并且将压缩包单独放入一个文件 。

2.进入CS目录，并且给予相关权限

chmod 777 *

3.查看ip地址 ifconfig

运行cs，设置服务器ip和密码 ./teamserver.sh 192.168.0.108 admin（admin是密码）

注意服务的端口号为，这里默认的是 27000

4.回到win10 解压后打开文件夹 找到CobaltStrike_CN.vbs 或者CobaltStrike_eN.vbs打开

IP为服务器（kali）的IP
端口号 在CS启动的时候可以看见端口号
用户名 为登录之后给别人看的名字，可以瞎写
密码 为之前设置的密码

登录成功界面

二、CS简单使用

1.点击左上方CobaltStrike选项——>在下拉框中选择 Listeners ——>在下方弹出区域中点击add 建立一个监听器

name随便取，HTTP Hosts点击旁边的加号，HTTP Port（C2）随便取

HTTP Hosts 是shell反弹的主机，就是kali

HTTP Port（C2） C2监听的端口

2.生成钓鱼网站

Attacks –> Web Drive-by –> Clone Site

Cone URL 选择要克隆的网站网址

勾上监听键盘选项

成功后会生成一个url,点击url,并且任意敲击键盘

3.生成木马

选择 Attacks –> Packages –> Windows Executable

Listener 选择想要监听的

x64也打上勾

然后点击Generate保存文件

4.双击文件，CS中就会显示上线（用虚拟机好一点，我用的物理机-。-）

5.渗透

先右键靶机，把session中sleep的值改为0

6.右键靶机点击interact渗透
